# Omarchpods

A fork of [MagicPodsCore](https://github.com/steam3d/MagicPodsCore) with a TUI, designed for Omarchy.

## Install

```
git clone git@github.com:tomycostantino/omarchpods.git && \
cd omarchpods && \
bash install.sh
```
